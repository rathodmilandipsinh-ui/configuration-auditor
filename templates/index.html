Platform Capabilities

Five scanners.
One unified result.

WHOIS & Domain Intel
Registrar, creation date, expiry countdown, nameserver delegation — catch domains headed for expiry before attackers do.
SSL / TLS Inspection
Certificate issuer, subject, validity window, days remaining, and a four-tier expiry status: VALID → WARNING → CRITICAL → EXPIRED.
DNS Record Mapping
A, AAAA, MX, TXT, and NS records resolved in parallel. SPF and DMARC absence flagged automatically.
Security Header Audit
HSTS, CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy checked and scored by impact weight.
Technology Fingerprint
50+ signatures across servers, CMS platforms, JS frameworks, analytics, CDN, and hosting — version-aware where possible.
AI Security Report
GPT-4o synthesises all findings into an executive summary, prioritised vulnerabilities, and step-by-step remediation guidance.
How It Works
01
Enter a domain
Type any domain name — no protocol needed. The engine normalises the input and validates it before scanning.
02
Parallel scan
All five scanners run concurrently via a thread pool, so a full audit completes in the time of the slowest single check.
03
Scored report
A weighted 0–100 score, per-scanner breakdown, AI narrative, and prioritised fix list — ready to act on immediately.

Biography

I am an Information Technology student with a strong interest in cybersecurity, Python development, and web application security. ConfigAudit is a project developed to automate security assessments by analyzing SSL certificates, DNS records, WHOIS information, HTTP security headers, and web technologies. The goal of this project is to demonstrate practical cybersecurity and backend development skills using Flask and Python.
